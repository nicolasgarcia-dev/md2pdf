    Custom CSS — How it works

    Your CSS is loaded after base.css, so you can override any default style. It applies to both the live preview and the downloaded PDF.

    Available fonts (pre-installed on the server)

    Noto Sans Noto Serif Inter JetBrains Mono

    Key selectors

    Selector What it controls
    html, body Font family, text colour, page background (use with @page { background-color } for full coverage)
    h1 – h4 Heading size, weight, colour, border
    a Link colour and decoration
    p Paragraph spacing and alignment
    blockquote Quote block border, background, italic
    pre, .codehilite Code block background and font
    code Inline code chip colour
    table, th, td Table borders and cell padding
    nav.toc Table of contents box
    main.document Main content wrapper

    Useful PDF-specific properties

    Property Effect
    body { padding: … } Page margins (e.g. 15mm 20mm) — default is 20mm 18mm
    page-break-inside: avoid Keeps an element on one page
    page-break-after: avoid Prevents a page break after the element
    text-align: justify Justified paragraph text

    Example — full page background colour

    One rule is enough — the server automatically extends the colour to the margin area so the entire page (including gutters) gets the same fill. No need to write an @page rule yourself. 

    /* Warm cream paper */
html, body { background: #fdf6e3; }

/* Slate dark page */
html, body { background: #1e1e2e; color: #cdd6f4; }
h1, h2, h3 { color: #89b4fa; }
a { color: #89dceb; }

    Example — dark academia style

    body {
    font-family: "Noto Serif", Georgia, serif;
    color: #1c1410;
    background: #fdf8f0;
}

h1, h2, h3 {
    font-family: "Inter", sans-serif;
    color: #7b2d00;
    letter-spacing: -0.02em;
}

h1 { border-bottom: 2px solid #c9834a; }

a { color: #7b2d00; text-decoration: underline; }

blockquote {
    border-left: 4px solid #c9834a;
    background: #fdf0e0;
    font-style: italic;
}

    Tips

    • Changes apply to the preview in real time. The PDF is only generated on download.
    • To colour the full page background, write html, body { background: #colour; } — the server automatically fills the margin area with the same colour so the entire page looks uniform.
    • Your CSS is saved automatically in the browser between sessions.
    • To turn a custom style into a permanent server-side theme, copy it to backend/app/assets/themes/my-theme.css and register it in themes.py — see docs/THEMES.md.
